# 🏪 Sistema POS Ferretero

Bot de Telegram para Punto de Venta en ferreterías.

## Características
- ✅ Ventas con precios personalizados
- ✅ Descuentos por unidad y al total
- ✅ Precios sugeridos por cantidad
- ✅ Reportes financieros
- ✅ Control de inventario
- ✅ Movimientos de productos
